左クリック -> 該当なし

マウスホイール -> 拡大・縮小

キー操作:
R -> 染色体を反転(上側の黄色枠内)
T -> 染色体を入れ替え(黄色枠間)
Y -> コンティグを反転(上側の水色枠内)
U -> コンティグを入れ替え(水色枠間)
N -> 1回目 -> コンティグを複数反転させる領域の開始部位指定(上側の水色)
　　 2回目 -> 上側水色の位置までの範囲のコンティグを反転
M -> 1回目 -> コンティグを複数ブロック間で交換するための領域1の開始部位指定(上側の水色)
　　 2回目 -> 上側水色の位置までの範囲のコンティグで領域1を確定
　　 3回目 -> 領域2の開始部位指定
　　 4回目 -> 領域2を確定し、領域1と2を入れ替える。
E -> 1回目 -> 複数のコンティグの染色体名を変更する領域の開始部位指定(上側の水色)
　　2回目 -> 複数のコンティグの染色体名を変更
